- 本项目基于 [RuoYi-Vue](http://ruoyi.vip/) 进行二次开发，高校工作室的工作流应用场景。
- 设计时没有考虑清楚项目耦合度高，对标毕设项目
- 推荐一个工作流项目：https://gitee.com/KonBAI-Q/ruoyi-flowable-plus/releases/tag/v0.6.0
